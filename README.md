### 작품 게시물 서비스
#### 기간 & 팀: (23.01~23.06) / 프론트 2명 + 백엔드 2명(팀장)
#### 기술스택: Java, SpringBoot, SpringData JPA, MySQL

<br>

### 구현 기능
- 작품, 만족도 API 구현, MVC 패턴 적용
- 인덱스를 적용해 페이징 구현, 특정 시작 위치 속도 개선
- AWS S3 Presigned URL 로 이미지 요청 구현, 서버 부하 감소
- AWS EC2와 Nignx로 인프라 구성, Reverse Proxy Server 이용해 무중단 배포 적용

<br>
  
### 특이 사항
- 협업을 위해 문서화 작업, 테스트 코드 작성 
- 코드 리팩토링 진행 중
